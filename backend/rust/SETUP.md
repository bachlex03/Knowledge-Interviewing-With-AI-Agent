# SETUP rust environment

description: this folder contains the setup guide for rust environment

## Install Rust

ref docs: `https://doc.rust-lang.org/book/ch01-01-installation.html`

```bash
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

**Check installation**

```bash
$ rustc --version
```
