# TypeScript Topic

---

## Scope
This topic covers TypeScript interview preparation, from core concepts and type safety to advanced utility types and architectural patterns.

## Tracks
- **Foundation**: Basic syntax, common types, interfaces, type aliases, and fundamental type system concepts.
- **Advance**: Generic types, utility types, conditional types, mapped types, decorators, and complex type manipulations.
- **Pitfalls**: Common mistakes like `any` abuse, `!` non-null assertion misuse, over-engineering types, and runtime vs compile-time behavior misunderstandings.
