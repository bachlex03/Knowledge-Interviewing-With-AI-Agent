# Bloom's Taxonomy Question Generator

Use this skill to create question sets for a topic using a simplified 5-level Bloom's Taxonomy structure.

## The 5 Levels

1. **Remembering**
   Focus on recall of facts, terms, definitions, or basic concepts.
   Useful verbs: define, list, identify, name.

2. **Understanding**
   Focus on explanation, summary, interpretation, or comparison.
   Useful verbs: explain, describe, summarize, discuss.

3. **Applying**
   Focus on using knowledge in practice, solving problems, or implementing ideas.
   Useful verbs: apply, use, solve, demonstrate.

4. **Analyzing**
   Focus on breaking a system into parts, comparing alternatives, or identifying causes and relationships.
   Useful verbs: analyze, compare, contrast, categorize.

5. **Evaluating**
   Focus on judgment, tradeoffs, critique, or defended opinions.
   Useful verbs: evaluate, justify, critique, defend.

## Instructions

1. Identify the topic the user wants to explore.
2. Generate questions for each of the 5 Bloom levels.
3. Provide a brief expected answer or key points for each question.
4. If the answer benefits from code, include a real code example in a fenced code block.
5. If another active skill or repository rule requires a specific output format, apply that format in the final answer.
